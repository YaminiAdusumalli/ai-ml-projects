# Resume Matcher v2

An AI-powered resume matching system that compares candidate resumes against job descriptions using NLP and embedding-based similarity scoring.

## Features
- Resume parsing
- Job description parsing
- Semantic similarity scoring
- AI-based ranking system
- Embedding-based similarity matching
- Recruiter dashboard ready architecture

## Tech Stack
- Python
- NLP
- Embeddings
- FastAPI
- LLM APIs
- Vector Search

## Architecture
- Resume → Text Extraction
- JD → Embedding Generation
- Cosine Similarity Matching
- Score Ranking Engine

## Use Case
Helps recruiters automatically rank candidates based on job fit using AI.
